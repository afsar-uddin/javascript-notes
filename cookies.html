cookies

Definition

An HTTP cookie (web cookie, browser cookie) is a small piece of data that a server sends to the user's web browser. The browser may store it and send it back with later requests to the same server. Typically, it is used to tell if two requests came from the same browser â€” keeping a user logged-in, for example. It remembers stateful information for the stateless HTTP protocol.

Cookies are mainly used for three purposes:

1) Session management Logins, shopping carts, game scores, or anything else the server should remember

2) Personalization User preferences, themes, and other settings

3) Tracking Recording and analyzing user behavior
