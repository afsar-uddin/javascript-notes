setTimeOut()

Definition

1) The setTimeout() method calls a function or evaluates an expression after a specified number of milliseconds.

2) The global setTimeout() method sets a timer which executes a function or specified piece of code once the timer expires.


function doSomething() {
    console.log('I am coding JavaScript');
}
setTimeout(doSomething, 6000);
