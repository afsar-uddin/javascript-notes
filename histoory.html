Window History

Definition

History The Window.history read-only property returns a reference to the History object, which provides an interface for manipulating the browser session history (pages visited in the tab or frame that the current page is loaded in).

Hints
1) history.back(); history.go(-1); both are same.
2) history.forward(); history.go(1) both are same.
history.back() means that one stap back and histoory.forward() means that one stap forward. history.go(negative or positive number), if negative number it means the go back and if positive number means that go forward.
