array

Definition

1) An array is a special variable, which can hold more than one value at a time.

2) The JavaScript Array class is a global object that is used in the construction of arrays; which are high-level, list-like objects.

Example 1 
let fruits = ['oranges', 'grapefruits', 'mandarins', 'limes', 'apricots']

console.log(fruits.length)
Example 2 
let numbers = [1,2,3,4,5,6]

console.log(numbers.length)
