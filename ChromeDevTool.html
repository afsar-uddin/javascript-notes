Console API

Definition

The Console API provides functionality to allow developers to perform debugging tasks, such as logging messges or the values of variables at set points in code, or timing how long an operation takes to complete.

The Console Utilities API contains a collection of convenience functions for performing common task: selelecting and inspecting DOM elements, displaying data in readable format, stopping and starting the profiler and monitoring DOM events.
